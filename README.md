# ESP32 Smart Battery Tester

This project measures 1.5V cell voltage using ESP32 and displays battery percentage on a 16x2 LCD.

## Features
- Real-time voltage display
- Battery percentage indicator
- Low voltage warning
- High voltage warning
- Startup animation
- Calibration support

- ESP32
- 16x2 LCD
- 2x 330kÎ© resistors
- Jumper wires
